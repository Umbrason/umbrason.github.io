Unity Packages

Animation System

Purely code-driven Animator component replacement using the 'Playables' API. Supports layering and 2D motion blending!

ASCIIFY Image Effect

URP renderfeature that replaces the output with specified symbols.

.Blend Import Fixer

Fixes missing animations and bakes the coordinate system conversion by modifying the import script.

Character Controller
Template

Template for a Rigidbody based (first-person) Character Controller using my Velocity Controller

Combat Script

Custom interpreted scripting language using ANTLR4. Intended for user generated content like custom card effects or abilities.

Interactable Shadergraph

Custom Shadergraph Node & Components to get the positions of 'Interactor' objects from within a shader.

Lightweight Serialization

Utility framework for semi-manual gamestate serialization, either explicitly or automatically via reflections.

Mesh Instancing Utils

Utility functions & components for drawing a lot instances of a single mesh like grass (GPU Instancing)

Orbit Camera Controller

Player Controller rotating a Camera around a pivot. Supports motion, rotation & zoom with configurable constraints

Package System

Skeleton structure for a UGC package system with dependencies

Procedural Mesh Utils

Contains a class to simplify mesh generation and merging.

Runtime Editor

Runtime Editor Framework with dock- and resizeable windows and saveable layouts.

Runtime (De-)Serialization

Utility classes for Runtime (De-)Serialization, mainly for rigged 3D models using the Collada file format.

Scriptable Object Libraries

Adds Singleton-esque libraries for icons, prefabs etc. Can generate a field per non-abstract inherited type using reflections

Themed UI

Adds 'Themed' variants of all major UI components, which update to reflect a centralized pallette for easy UI color themes.

Git Package Hub

In-Engine explorer for my Github Packages. Fetches available packages and allows for installation with one click.

Umbrasons Utilities

Harmonic Springs, Automagic Lazy References and more!
Usefull utilities I usually use in Unity.

Unity Kanban

Simple Kanban window and asset for the Unity Editor. Allows you to quickly keep track of tasks in engine.

Velocity Controller

Controlls the velocity of a Rigidbody by layering 'Velocity Overrides'. Allows for blending of multiple effects e.g. wind
